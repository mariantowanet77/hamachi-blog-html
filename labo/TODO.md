# Labo — これからやることメモ

_更新: 2026-07-24_

## これは何
`labo/` は記事とは別の **実験用フォルダ**。WebGL アニメや思いついた処理を
テキトーに置いていく遊び場。目標イメージは https://no.meets.ltd/ みたいな
華やかな WebGL 演出。

## 決めたこと（方針）
- **入口**: ヘッダーのハンバーガーメニューに「🧪 Labo」リンクを追加する。
  - **ゲート無し**（＝誰でも入れる）で確定。
    ※ 静的サイト（S3 / CloudFront）なので**バックエンド認証は付けない**方針。
      （「ログイン」画面にすると本物の認証ではなくクライアント判定になるため見送り）
  - 実装は各ページの HTML を触らず、`common/site.js` の `initNav()` で
    **全ページに一括注入**するのが DRY。
- **`labo/index.html` の中身**: 依存ライブラリ無しの **全画面 WebGL フラグメントシェーダー**。
  - 手法: fbm ノイズ → ドメインワープ → iq の余弦カラーパレット、の王道演出。
  - 自己完結（インライン CSS / JS）。共通の `/common/site.js` だけ読み込んでメニューを共有。
  - 暗い没入デザイン。ヘッダーはガラス風(blur)、ハンバーガーの三本線は白に上書き。
  - 現状は「準備中」プレースホルダを置いてあるだけ。← これを本実装に差し替える。
- **ひ♡み♡つ の強調**: `common/style.css` の `.secret-link` を
  「大きく・脈動する目立つピル」に変更する。
  - 理由: ポートフォリオで見せたときにスルーされないように。
