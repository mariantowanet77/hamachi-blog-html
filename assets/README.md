# assets フォルダ

ここには **画像などの素材** を入れます（HTML/CSS/JS などのコードは `web` 側）。

フォルダ構成は `web` と同じ形にそろえてあります。
記事やページの画像は、`web` の対応するフォルダと **同じ名前のフォルダ** に入れてください。

| web 側のページ                         | 画像を置く場所                          |
| -------------------------------------- | --------------------------------------- |
| `web/index.html`（トップ）             | `assets/`（直下）                       |
| `web/common/`（共通CSS・JS）           | `assets/common/`（ロゴ・favicon など）  |
| `web/about/`                           | `assets/about/`                         |
| `web/pastarticles/`                    | `assets/pastarticles/`                  |
| `web/pastarticles/hello-world/`        | `assets/pastarticles/hello-world/`      |

`web` に新しいフォルダを作ったら、ここにも同じ名前のフォルダを作ってください。
